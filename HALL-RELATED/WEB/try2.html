24/7 Online Storage Setup

To enable 24/7 online storage with Google Sheets:

  1. Create a new Google Sheet here
  2. Name it "King of Cox Bookings"
  3. Click on "Extensions" → "Apps Script"
  4. Delete the default code and paste the code from the "Google Apps Script Code" section below
  5. Click the save icon and name your project "KingOfCoxAPI"
  6. Click "Deploy" → "New deployment"
  7. Select "Web app" as the type
  8. Set "Execute as" to "Me" and "Who has access" to "Anyone"
  9. Click "Deploy" and copy the provided URL
  10. Paste the URL in the script below where it says "YOUR_GOOGLE_SCRIPT_URL"

Note: This will enable real online storage instead of browser-only storage.
